# 4.5 VBA的图片接口怎么用？怎么实现点哪儿插哪儿
* 注意： 9.4以上才有图片插入VBA接口
```vb
	Sub pic_insert()
		Dim fn
		fn = Application.GetOpenFilename()
		Application.COMAddIns("ESClient10.Connect").Object.AddPicture fn, 1, ActiveCell.Row, ActiveCell.Column
	End Sub
```
