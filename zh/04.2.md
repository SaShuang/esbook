# 4.2 VBA提取中英文代码

## @heming
```sql
Function getchn(MyValue As Range)
	Dim i As Integer
	Dim chn As String
	For i = 1 To Len(MyValue)
		If Asc(Mid(MyValue, i, 1)) < 0 Then
		chn = chn & Mid(MyValue, i, 1)
		End If
	Next
	getchn = chn
End Function

Function yw(str As String)
	 With CreateObject("vbscript.regexp")
	     .Global = True
	     .Pattern = "[^a-zA-Z]"
	     yw = .Replace(str, " ")
	End With
End Function
```
## 补充
 * getchn获取中文
 * yw获取英文
